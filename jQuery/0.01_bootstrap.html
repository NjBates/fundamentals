Bootstrap Set-up

Getting Started

Go to the Bootstrap Website and click Download Bootstrap. Then you have a couple options:

  1. Use the CDN links provided (you must be online for these to work)
  2. Download the actual files and put them in your project folders (this allows you to work offline)
  3. Click 'Customize' at the top of the page and build your own bootstrap package based on what you need for your particular project.

Here are the next steps you should take:

  1. Place the minified CSS link in the head
  2. Place the minified JavaScript link in the head
  3. You can delete or keep the optional link depending on whether or not you need it
  4. Lastly, it's super important to include a link to jQuery, because Bootstrap has a dependency on jQuery. (You can go to the jQuery website and copy the latest stable minified version and paste it right above the minified JavaScript bootstrap link in the head.

You're now ready to start creating a responsive website!
